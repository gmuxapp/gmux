## Install

<Tabs>
  <TabItem label="macOS (Homebrew)">
    ```bash
    brew install gmuxapp/tap/gmux
    ```
  </TabItem>
  <TabItem label="Linux">
    ```bash
    curl -sSfL https://gmux.app/install.sh | sh
    ```
    [What does this script do?](/install.sh)
  </TabItem>
</Tabs>

Or download both binaries (`gmuxd` + `gmux`) from [GitHub Releases](https://github.com/gmuxapp/gmux/releases). On Windows, the Linux instructions work inside WSL.

## Run

```bash
gmux pi                    # launch a coding agent
gmux pytest --watch     # or any command
gmux make build
```

Open **[localhost:8790](http://localhost:8790)**. Sessions appear in the sidebar grouped by project. Click one to attach a live terminal.

`gmux` auto-starts the daemon (`gmuxd`) on first run — there's nothing else to set up. For daemon commands, run `gmuxd -h`.
